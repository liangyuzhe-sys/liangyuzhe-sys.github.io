# Yuzhe Liang Personal Website

This repository is a static GitHub Pages site for Yuzhe Liang.

## Local Preview

Open `index.html` directly in a browser.

## Deploy To GitHub Pages

1. Create a public repository named `YOUR_GITHUB_USERNAME.github.io`.
2. Push these files to the repository's `main` branch.
3. Open `https://YOUR_GITHUB_USERNAME.github.io`.

If you use a normal repository name instead, enable GitHub Pages in repository settings and choose the `main` branch root as the source.
